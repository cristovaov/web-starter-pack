# web-starter-pack

> A cli to start new web projects in seconds

## Usage

With `npx` (recommended):
```bash
npx web-starter-pack <project-folder>
```
With `npm <= 5.1`:
```bash
npm install -g web-starter-pack
web-starter-pack <project-folder>
```
